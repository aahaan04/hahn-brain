# Hahn Agency Brain

A RAG-powered chatbot that answers questions about Hahn Agency using content scraped directly from their website. Built as a job application project to demonstrate practical AI engineering applied to Hahn's real brand, services, and team.

---

## What it does

- Scrapes 12 pages from hahn.agency (services, industries, about, work, and more)
- Chunks and embeds the content using OpenAI's `text-embedding-3-small`
- Stores all vectors locally in a ChromaDB collection
- Serves a Streamlit chat interface that retrieves the most relevant chunks on every question and feeds them to `gpt-4o-mini`
- Answers only from real Hahn content, never hallucinates, and warmly redirects to Hahn's contact details when information is not available

---

## Tech stack

| Layer | Tool |
|---|---|
| Scraping | crawl4ai + Playwright (Chromium) |
| Embedding | OpenAI text-embedding-3-small |
| Vector store | ChromaDB (local persistent) |
| Chat model | gpt-4o-mini |
| UI | Streamlit |
| Env management | python-dotenv |

## Setup

### 1. Clone the repo

```bash
git clone git@github.com:aahaan04/hahn-brain.git
cd hahn-brain
```

### 2. Create a virtual environment

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install crawl4ai openai chromadb streamlit python-dotenv
```

### 4. Install the Playwright browser (required by crawl4ai)

```bash
python -m playwright install chromium
```

### 5. Add your OpenAI API key

Create a `.env` file in the project root:

```
OPENAI_API_KEY=sk-...
```

## Running the pipeline

Run these three steps in order. Steps 1 and 2 only need to be run once (or whenever you want to refresh the content).

### Step 1 - Scrape Hahn Agency

```bash
python scrape.py
```

Crawls 12 pages from hahn.agency and saves the content to `hahn_content.json`.

### Step 2 - Embed and store

```bash
python embed.py
```

Chunks each page into 500-word blocks (50-word overlap), embeds them with OpenAI, and stores them in a local ChromaDB collection called `hahn`. Processes in batches of 50 to stay within API rate limits.

### Step 3 - Launch the chat UI

```bash
streamlit run app.py
```

Opens the chatbot at `http://localhost:8501`.

## How it works

1. The user types a question.
2. The question is embedded with `text-embedding-3-small`.
3. ChromaDB returns the 8 most semantically similar chunks.
4. Those chunks are passed as context to `gpt-4o-mini` along with the system prompt.
5. The model answers strictly from the retrieved context. If a question has multiple parts, it answers every part it can and redirects to Hahn's contact info for anything not covered.

## Notes

- `venv/` and `.env` are excluded by `.gitignore`.
